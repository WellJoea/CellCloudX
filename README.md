# CellCloudX - spatial multi-omics registration tool
The [**CellCloudX**](https://github.com/WellJoea/CellCloudX.git) is a Spatial transcriptome cell registration tool -.<br/>
You can ues most of the estimator in sklearn in MLkit and implement auto standardization, feature selection, Fitting and Prediction.<br/>

___
## Installation
### Dependencies
<pre><code>   
        'numpy >= 1.16.4',
</code></pre>
### User installation
- download: https://github.com/WellJoea/CellCloudX.git
- cd CellCloudX
- pip install .
